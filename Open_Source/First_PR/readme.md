# Git-learning-repo

This folder is basically made for a complete beginner make his/her first contribution and get acquainted with the open source ♥‿♥. You are welcomed to fork this repo and follow the steps mentioned below and come up with a PR, I would be happy to merge it (~˘▾˘)~

- Fork the repo
- Clone it on local (git clone <your_forked_repo_url>)
- Make a new branch from `main` (git checkout -b <new_branch_name>)
- checkout to that new branch [done in previous step]
- Make the required changes (Maybe add a file similar to `demo.html`(https://github.com/Kajol-Kumari/winter-of-contributing/Open_Source/First_PR/about_yourself/demo.html) containing the information about who you are, what you do etc :P)
- Add to staging (git add .)
- Commit it. (git commit -a -m "<your_commit_msg>")
- Push it.(git push origin <your_branch_name>)
- Make a PR.
